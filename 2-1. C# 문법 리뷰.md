## 1. 변수와 데이터 타입

변수는 데이터를 저장하는 상자라고 생각하면 됩니다.  
데이터 타입은 변수 안에 어떤 종류의 값을 넣을 수 있는지를 결정합니다.

```csharp
int score = 0;
float speed = 5.5f;
bool isDead = false;
string playerName = "Haedal";
```

| 타입 | 설명 | 예시 |
|------|------|------|
| int | 정수 | 체력, 점수 |
| float | 소수점 | 속도, 위치 |
| bool | 참/거짓 | 점프 중인지 여부 |
| string | 문자열 | 플레이어 이름, 로그 메시지 |

📖 공식문서: [C# Data Types - Microsoft Docs](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/builtin-types/built-in-types)

## 2. 연산자

연산자는 값을 계산하거나 비교하는 데 사용됩니다.

```csharp
int hp = 100 - 20;         // 뺄셈
bool isLow = hp < 30;      // 비교
score += 10;               // 누적
isJumping = !isGrounded;   // 논리 부정
```

| 연산자 | 의미 | 예시 |
|--------|------|------|
| +, -, *, / | 기본 사칙 연산 | `a + b` |
| ==, !=, <, > | 비교 | `a == b` |
| &&, \|\|, ! | 논리 연산 | `a && b` |

📖 공식문서: [C# Operators - Microsoft Docs](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/operators/)

## 3. 조건문 (if, switch)

조건문은 "이럴 경우에는 이렇게 해!"라는 분기 로직을 만듭니다.

```csharp
if (hp <= 0)
{
    Die();
}
else
{
    Recover();
}
```

```csharp
switch (weaponType)
{
    case "gun":
        Shoot();
        break;
    case "sword":
        Attack();
        break;
}
```

📖 공식문서: [if, else, switch - Microsoft Docs](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/selection-statements)

## 4. 반복문 (for, while, foreach)

같은 작업을 여러 번 반복하고 싶을 때 사용합니다.

```csharp
for (int i = 0; i < 10; i++)
{
    Debug.Log("Wave: " + i);
}

while (isPlaying)
{
    time += Time.deltaTime;
}

foreach (GameObject enemy in enemies)
{
    enemy.SetActive(false);
}
```

## 5. 클래스와 MonoBehaviour

Unity의 모든 게임 오브젝트는 **클래스**로 구성됩니다.  
`MonoBehaviour`는 Unity가 제공하는 기본 클래스입니다.

```csharp
public class Enemy : MonoBehaviour
{
    public int hp = 100;

    void Start()
    {
        Debug.Log("적 등장!");
    }

    void Update()
    {
        Move();
    }

    void Move()
    {
        transform.Translate(Vector3.left * Time.deltaTime);
    }
}
```

📖 공식문서: [MonoBehaviour - Unity Docs](https://docs.unity3d.com/ScriptReference/MonoBehaviour.html)

## 6. Unity 기본 함수 (생명주기)

| 함수명 | 호출 시점 | 용도 |
|--------|------------|------|
| `Awake()` | 가장 먼저 1회 | 초기 설정 |
| `Start()` | 시작 시 1회 | 게임 시작 준비 |
| `Update()` | 매 프레임 호출 | 이동, 입력 처리 |
| `FixedUpdate()` | 일정 시간마다 호출 | 물리 계산 |
| `LateUpdate()` | Update 후 호출 | 카메라 따라가기 등 |

📖 공식문서: [Execution Order - Unity Docs](https://docs.unity3d.com/Manual/ExecutionOrder.html)

## 8. 충돌 처리

```csharp
void OnCollisionEnter(Collision other)
{
    if (other.gameObject.tag == "Enemy")
    {
        TakeDamage(10);
    }
}

void OnTriggerEnter(Collider other)
{
    if (other.CompareTag("Item"))
    {
        CollectItem();
    }
}
```

## 9. 코루틴 (Coroutine)

## 11. 실습 예제: PlayerController.cs

```csharp
public class PlayerController : MonoBehaviour
{
    public float speed = 5f;
    public float jumpPower = 7f;
    private Rigidbody rb;

    void Start()
    {
        rb = GetComponent<Rigidbody>();
    }

    void Update()
    {
        float h = Input.GetAxis("Horizontal");
        transform.Translate(Vector3.right * h * speed * Time.deltaTime);

        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.AddForce(Vector3.up * jumpPower, ForceMode.Impulse);
        }
    }
}
```
